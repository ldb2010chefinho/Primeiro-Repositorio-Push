Git Push

O git push é o comando que envia os commits do seu repositório local para um repositório remoto (como o GitHub).

Sintaxe Basica do Comando:

git push origin <nome-da-branch>


Git Fetch

O git fetch baixa novos dados de um repositório remoto, mas não altera seus arquivos ou branches de trabalho.

Ele permite que você veja o que outros enviaram antes de mesclar ou extrair.
